Build and Run XFrames Docker
============================

Build
-----
From the xframes main directory:

    cd docker
    ./build-xframes-docker

Run
---
    ./run-xframes-docker

Stop Docker
-----------
    ./stop-xframes-docker

Enter Docker
------------
    enter-xframes-docker
    
Using XFrames Docker
====================

Run notebook
------------
Browse to localhost:8000

View Documentation
------------------
Browse to localhost:7777
